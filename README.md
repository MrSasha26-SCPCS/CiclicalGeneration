# CiclicalGeneration
SCP: Classified Site plugin

Плагин, увеличивающий количество перекрёстков во всех зонах, что приводит к созданию зацикленных участков. 

Некоторые параметры можно менять через конфиг:
* **lcz_4rooms_count** - количество перекрёстков в ЛЗС;
* **hcz_4rooms_count** - количество перекрёстков в ТЗС;
* **ez_4rooms_count** - количество перекрёстков в офисах.

*Примечание:* если установить для вышеперечисленных параметров слишком большие значения (более 8), то при генерации могут возникать ошибки. В этом случае можно либо уменьшить значения до стандартных (6-8), либо увеличить значение стандартного параметра `generator_random`.
